# VLM Model Ayarı + Hisse Analizi Görsel Geliştirme

## A. VLM Model Seçimi (Ayarlar)

### Backend
- `routers/admin.py` → `GET /api/admin/vlm-config` — `get_setting("vlm_model")` döner (prediction-config deseni)
- `routers/skill.py` `analyze_kline` → `get_setting(db, "vlm_model")` oku, `kline_chart.run(ticker, period, model=...)` geç
- `skills/kline_chart.py` `run()` → `model: Optional[str] = None` parametresi al, `_vlm_analyze`'a geç
- `llm_bridge.generate_vision()` zaten `model` parametre alıyor — değişiklik yok
- `admin_service.get_setting/set_setting` mevcut — yeni kod yok

### Frontend
- `frontend/app/ayarlar/VLMTab.tsx` — text input + hazır öneri butonları:
  - Hazır: `openai/gpt-4o-mini`, `openai/gpt-4o`, `claude-3-5-sonnet-20240620`, `gemini/gemini-1.5-pro`, `ollama/llava`
  - Kaydet → `POST /api/admin/settings` `{key: "vlm_model", value: "..."}`
- `ayarlar/page.tsx` → 5. sekme "VLM"

## B. Hisse Analizi Görsel Geliştirme

### Backend
- `schemas/analysis.py` `StockAnalysisResult`'a 3 alan ekle:
  ```python
  price_history: list[dict] = []  # [{date, open, high, low, close, volume}] son 60 gün
  scores: dict = {}  # {fundamental, sentiment, risk, composite}
  position_pl: Optional[dict] = None  # {cost_total, current_total, pl, pl_pct}
  ```
- `skills/stock_analysis.py` `run()` zenginleştir:
  - `history` zaten `safe_ticker_history` ile çekiliyor → son 60 gün OHLCV `price_history`'e paketle
  - `candidate` dict'ten `scores` çıkar
  - `position_pl` zaten `_format_pl` ile hesaplanıyor — döndür

### Frontend
- `lib/api.ts` `StockAnalysisResult` tipi güncelle — `price_history`, `scores`, `position_pl`
- `SkillPanel.tsx` stock tab sonucu → zengin dashboard:
  1. **Fiyat grafiği** — `PriceChart.tsx` (lightweight-charts) yeniden kullan, son 60 gün OHLCV + SMA20 overlay
  2. **Skor bar'ları** — fundamental/sentiment/risk/composite 0-100 yatay bar (renk: yeşil>60, sarı 40-60, kırmızı<40)
  3. **Bias gauge** — 乖离率 % görsel (büyük sayı, renk)
  4. **P/L kartı** — pozisyon varsa maliyet/güncünlük/K-Z renkli
  5. **İşlem planı** — entry/stop(-8%)/target(+10%) yatay bar
  6. **Markdown** — alt kısımda haber/özet (korunur)

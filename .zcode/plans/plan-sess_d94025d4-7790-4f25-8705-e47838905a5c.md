# Çoklu Portföy Yönetimi — BIST + US (NASDAQ+DJIA)

## Hedef
Sistemde 2 ayrı otonom portföy: BIST (84 ticker) + US (NASDAQ 64 + DJIA 30, çakışanlar tek say ~85). Her portföye $10.000 başlangıç. 2 ayrı AutonomousAgent paralel çalışır, ayrı al-sat kararları, ayrı pozisyonlar.

## Mimari

### Yeni: `Portfolio` modeli (`models/portfolio.py`)
```python
class Portfolio(Base):
    __tablename__ = "portfolios"
    id = Column(Integer, primary_key=True, index=True)
    slug = Column(String, unique=True, index=True)           # "bist" | "us"
    display_name = Column(String)                            # "BIST Portföyü"
    exchanges = Column(JSON, default=list)                   # ["BIST"] | ["NASDAQ","DJIA"]
    cash = Column(Float, default=10000.0)
    max_positions = Column(Integer, default=8)
    max_per_position_pct = Column(Float, default=0.25)
    is_active = Column(Boolean, default=True)
    created_at, updated_at
```
`VirtualBalance` deprecate (yeni kod `Portfolio.cash` kullanır; legacy `balance_service.get_balance` uyumluluk için `Portfolio`'dan okur).

### FK eklemeler (mevcut modellere `portfolio_id` nullable)
- `PortfolioPosition.portfolio_id` → `portfolios.id` (nullable — null = legacy)
- `TradingDecision.portfolio_id` → `portfolios.id`
- `BalanceTransaction.portfolio_id` → `portfolios.id`

### `autonomous_agent.py` — parametre portföy bilinci
```python
class AutonomousAgent:
    def __init__(self, portfolio_slug: str = "bist", model="ensemble-light"):
        self.portfolio_slug = portfolio_slug
        cfg = PORTFOLIOS[portfolio_slug]
        self.max_positions = cfg.get("max_positions", 8)
        self.max_per_position_pct = cfg.get("max_per_position_pct", 0.25)
        ...
    # get_portfolio(db) → sadece o portföyün pozisyonları + Portfolio.cash
    # execute_buy/sell → PortfolioPosition(portfolio_id=self.portfolio_id)
    # _log_decision → TradingDecision(portfolio_id=self.portfolio_id)
    # think_and_act(exchanges) → exchanges parametresi PORTFOLIOS[slug]["exchanges"]
```

### `balance_service.py` — portföy-bilinçli
```python
def get_balance(db, portfolio_id=None) -> Portfolio  # Portfolio.cash döner
def record_position_opened(db, position_id, cost, ticker, portfolio_id)
def record_position_closed(db, position_id, proceeds, ticker, portfolio_id)
def reset_balance(db, portfolio_id, starting_cash=10_000)
def reset_portfolio(db, portfolio_id)  # o portföyün pozisyon+decision+tx sil
```

### `scheduler.py` — 2 paralel job
```python
def _run_autonomous_bist_sync():
    AutonomousAgent(portfolio_slug="bist").run()  # exchanges config'ten

def _run_autonomous_us_sync():
    AutonomousAgent(portfolio_slug="us").run()

scheduler.add_job(_run_autonomous_bist_sync, IntervalTrigger(minutes=30), id="autonomous_bist")
scheduler.add_job(_run_autonomous_us_sync, IntervalTrigger(minutes=30), id="autonomous_us")
```

### `routers/autonomous.py` — portföy parametresi
- `GET /api/autonomous/portfolio?portfolio_id=bist|us`
- `POST /api/autonomous/run?portfolio_id=bist|us`
- `GET /api/autonomous/decisions?portfolio_id=bist|us`
- `POST /api/autonomous/schedule?portfolio_id=...` (her portföy ayrı schedule)

### Reset endpoint'leri güncelle (`routers/admin.py`)
- `POST /api/admin/reset/portfolio?portfolio_id=bist` — sadece BIST portföyünü sıfırla
- `POST /api/admin/reset/portfolio?portfolio_id=us` — sadece US
- `POST /api/admin/reset/portfolio` (portfolio_id yoksa) — tüm portföyler
- `/reset/reports` ve `/reset/all` aynen (raporlar portföy-bağımsız)

### Frontend
- `lib/api.ts` — `getAgentPortfolio(portfolioId)`, `runAgent(portfolioId)`, `getAgentDecisions(portfolioId)`
- `AgentPortfolioCard.tsx` — portföy selector (BIST/US tabs), her sekmede ayrı portföy özeti + kararlar
- `Ayarlar > SİSTEM SIFIRLAMA` — portföy seçici ile "BIST sıfırla" / "US sıfırla" / "Tümünü sıfırla"

3. `trading_decisions.portfolio_id` ekle
